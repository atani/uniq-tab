# uniq-tab

重複タブの防止と整理、GitHub PR自動分割を行うChrome拡張機能。

## 機能

### 重複タブ防止

同じURLのタブを新しく開くと、既存のタブにフォーカスして新しいタブを自動で閉じる。URLのハッシュフラグメントや末尾スラッシュの違いは無視する。

### 既存タブの重複整理

ポップアップの「重複タブをまとめる」ボタンで、今開いている全タブをスキャンし、同一URLの重複を一括で閉じる。最後にアクセスしたタブを優先して残す。

### GitHub PR自動分割

GitHub PRの概要ページを開くと、Filesタブを隣に自動で開く。GitHub Enterpriseにも対応しており、オプション画面から対象ホストを追加できる。

## インストール

1. このリポジトリをcloneまたはダウンロード
2. Chromeで `chrome://extensions` を開く
3. 「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」からこのディレクトリを選択

## 設定

拡張機能アイコンのポップアップから各機能のON/OFFを切り替えられる。「詳細設定」からGitHub PR自動分割の対象ホストを管理できる。

## 必要な権限

| 権限 | 用途 |
|------|------|
| `tabs` | タブの検索・作成・フォーカス・削除 |
| `storage` | 設定の保存 |
